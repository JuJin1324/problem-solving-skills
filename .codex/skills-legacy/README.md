# Codex Legacy Skills

이 디렉터리는 더 이상 메인 워크플로우에서 사용하지 않는 레거시 스킬 보관소입니다.

## 포함 스킬
- `2w-brainstorm`
- `1h-agile-phase`
- `sprint-start`
- `sprint-complete`
- `us-complete`

## 운영 규칙
- 새 작업은 `.codex/skills/`의 Active 스킬을 우선 사용한다.
- 레거시 스킬은 과거 프로젝트 재현/참조가 필요할 때만 사용한다.
- 레거시 -> Active 매핑은 `docs/operations/skills-lifecycle.md`를 따른다.
