# Define 2W

## 목적
무엇/왜 우선 원칙을 지키면서 질문 수를 줄여 **실행 가능한 2W vN**을 빠르게 만든다.

핵심 원칙:
- `완벽한 2W`가 아니라 `실행 가능한 2W vN`
- 질문은 최소화하고, 모호성은 스프린트 학습으로 줄인다
- 사용자가 말한 방법(How)에서 무엇/왜를 역으로 도출한다
- 방법(How) 상세는 여기서 하지 않고 `/design-phase`로 넘긴다

## 이 스킬이 메인으로 담당하는 것
- 사용자 자유 발화 수집
- How -> What/Why 역추출 및 재서술
- 2W 정의 (무엇/왜)
- 제약 확인 (시간/협업/완성도)
- 성공 기준 1개, 경계 1개 확정
- 다음 단계 결정 (진행/보류/중단)

## 출력물
- `.agile/loops/loop-vN/define-2w.md`
- `.agile/loops/loop-vN/define-2w-case-study.md` (선택: 사례 연구 실행 시 필수)

템플릿:
- `templates/define-2w-vN.md`
- `templates/define-2w-case-study-vN.md`

참고 문서:
- `references/philosophy.md`
  - 2W를 스프린트 단위로 줄이는 이유가 헷갈릴 때
  - 사례 연구를 언제 켜고 끌지 판단 근거가 필요할 때
- `references/experience.md`
  - 유사한 2W 문제에서 어떤 역추출/게이트 판단이 유효했는지 빠르게 참고할 때

## 작동 방식

### 1단계. 준비
- `references/philosophy.md` 핵심 원칙 확인
- `.agile/loops/loop-vN/` 디렉터리 확인/생성

### 2단계. 단일 입력 수집
사용자에게 아래 한 가지를 묻는다.
- "지금 무엇을 하고 싶은지, 떠오르는 방식/이유/불편한 점까지 자유롭게 말해달라"

### 3단계. 입력 기반 What/Why 역추출
수집된 자유 입력을 바탕으로 AI가 What/Why를 직접 도출한다.
- 도출된 What: "실제로 해결하려는 문제" 1문장
- 도출된 Why: "지금 해결해야 하는 이유" 1문장

도출 방식:
- 방법(수단) 표현 제거
- 문제/영향/목표 상태 중심으로 재작성
- How 단서는 참고하되, 출력은 What/Why만 확정
- 사용자에게 1회 확인 질문으로 합의

### 4단계. 사례 연구 판단 게이트
먼저 AI가 추천안을 만들고, 사용자에게 실행 여부를 확인한다.
- AI 추천: 진행 권장 | 생략 권장
- 추천 근거: 1줄
- 사용자 선택: 진행 | 생략

게이트 결과:
- `생략`: 사례 연구 생략 후 5단계로 이동
- `진행`: 사례 연구 실행
  - 최대 3개 사례만 조사
  - 각 사례는 3줄 요약
  - 결과를 `.agile/loops/loop-vN/define-2w-case-study.md`에 기록
  - 2W 문서에는 핵심 반영점만 요약

### 5단계. 2W vN 문서화
`templates/define-2w-vN.md`로 `.agile/loops/loop-vN/define-2w.md`를 작성한다.
- 사용자 자유 입력 원문
- 무엇/왜 한 줄 요약
- 성공 기준 1개
- 경계 1개
- 가정/미확정 항목
- 사례 연구 게이트 결과(AI 추천 + 사용자 선택 + 근거)
- 사례 연구 실행 시 결과 파일 경로
- 진행/보류/중단 결정

### 6단계. 다음 단계 결정
- `진행(GO)`: `/design-phase`
- `보류(HOLD)`: 질문 1개만 추가 후 2W 재작성
- `중단(STOP)`: 문제를 분할하고 새 vN으로 재시작

## 안티패턴
- 사용자에게 What/Why를 직접 정답처럼 요구함
- 질문지를 별도 파일로 강제해 입력 마찰을 키움
- 질문지를 길게 늘려 답변 피로를 유발함
- 사례를 장문 리서치로 확장함
- 2W 단계에서 구현 설계로 진입함

## 완료 조건
- 단일 자유 입력 수집 완료
- What 1문장, Why 1문장 도출 완료
- 성공 기준 1개, 경계 1개 확정
- 사례 연구 게이트 기록 완료 (AI 추천 + 사용자 선택)
- 사례 연구를 실행했다면 `.agile/loops/loop-vN/define-2w-case-study.md` 작성 완료
- `.agile/loops/loop-vN/define-2w.md` 작성 완료
- 진행/보류/중단 결정 완료
