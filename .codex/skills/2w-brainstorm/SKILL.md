# 2w-brainstorm

## Goal
Keep conversations in the problem space first.

## Steps
1. Ask "What exactly is the problem?"
2. Ask "Why does this matter now?"
3. Capture constraints (time, resources, stakeholders).
4. Write a concise summary with unresolved questions.

## Deliverables
- `questions.md`
- `2w-brainstorm.md`
