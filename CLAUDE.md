# CLAUDE.md

이 프로젝트는 Claude Code skills를 활용해 문제를 애자일 방식으로 해결하는 실행형 워크스페이스입니다.

## 핵심 원칙
1. `How`보다 `What/Why`를 먼저 정리한다.
2. `How`는 범위와 단계(Phase)로 구조화한다.
3. 구조화된 계획을 Sprint로 실행한다.

## 권장 흐름
1. `2w-brainstorm`: 문제/목적/제약 정의
2. `1h-agile-phase`: Mermaid 범위 다이어그램 + Phase 설계
3. `sprint-start`: Sprint 계획 시작
4. `sprint-status`, `sprint-complete`, `us-complete`: 실행/완료 관리

## Claude Skill Location
- Claude 스킬 경로: `.claude/skills/<skill-name>/SKILL.md`
- 스킬별 산출물은 해당 문제 디렉터리와 `.agile/sprints/`에 기록한다.
