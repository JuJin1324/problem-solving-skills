# CLAUDE.md

이 파일은 Claude Code가 이 저장소에서 작업할 때 참고하는 운영 가이드입니다.

## 저장소 목적
- 5주 취업 준비 로드맵을 문서 중심으로 실행/관리합니다.
- Sprint 산출물은 `.agile/sprints/sprint-N/` 아래 단계 디렉터리에 기록합니다.

## 핵심 원칙
1. `How`보다 `What/Why`를 먼저 확정합니다.
2. 실행은 `Direction -> Delivery -> Learning` 파이프라인을 따릅니다.
3. 각 단계 산출물은 반드시 해당 단계 디렉터리에 저장합니다.
